viewmatrix.js

viewmatrix.js is a lightweight, unopinionated library for creating carousels.

Still in development, but you're welcome to test it and give feedback!

Examples GitHub Repo

Examples

Simple gallery

Basic gallery, with default options and no extra modules.

The core ViewMatrix experience only messes with classes, meaning all visual effects below are achieved using CSS. 

var example1 = new ViewMatrix('.example-1');

Infinite carousel

A carousel that wraps around itself when enough elements exist.

ViewMatrix makes sure the necessary elements get their classes, but with infinite enabled, it also makes sure the elements at the edges of the carousel have the opposite classes. 

var example2 = new ViewMatrix('.example-2', {
	infinite: true
});

Autoplaying gallery

A gallery that changes slides over time, until manually interacted with. 

var example3 = new ViewMatrix('.example-3');
var example3autoplay = new ViewMatrixAutoplay(example3, {
	interval: 3000
});

Vertical carousel "window", with swipe

Self-contained carousel, with CSS styles overriden to make it look vertical, and with "quick swipe" capabilities using the TouchSwipe module.

infinite is enabled to make sure the slides are put at the right place when navigating in the carousel's edges. preventDefault also makes sure the window doesn't move around while swiping on mobile. 

var example4 = new ViewMatrix('.example-4', {
	infinite: true
});
var example4touch = new ViewMatrixTouchSwipe(example4, {
	preventDefault: true,
	vertical: true
});
